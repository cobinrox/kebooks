Command Local or Remote Definition
clone remote Copies down a remote repo to your laptop.
status local Displays information about your local repo only.
remote update remote Updates your local ledger with new branch information from the remote repo.
branch local Displays branches that are part of your local repo.
checkout local Changes to or creates a new branch on your local repo.
add local Moves your local changes to the local staging area on your local repo
commit local Moves your local changes to the commit area on your local area and creates a commit ID.
pull remote Pulls down and attempts to merge in the latest version of a branch from the remote repo down to your local repo.
fetch remote Fetches remote information about the latest version of a branch from the remote down to your local repo for comparisson. Does NOT make any changes to your local repo.
push remote Pushes local branch from local repo up to the same branch on remote repo.

In the remaining sub-sections, you are encouraged to follow along on your laptop. You can refer to Appendix A: Installing Git which will walk you through installing Git on your laptop, setting you up with a Git account on GitHub, and creating a dummy repo on GitHub.
